# 恋はじめ（koi-hajime） - 共通コンテキスト

## プロジェクト概要
転職・就職を機に正社員になった（目指している）20代向けのマッチングアプリ比較アフィリエイトサイト。
「安定した仕事を手に入れたら、次は恋愛・婚活」という読者ジャーニーを捉える。

## ターゲット
- 20代フリーター・未経験が転職活動中〜正社員になったばかりの層
- 「仕事が安定したら出会いを探したい」と思っている人
- challenge-tensyokuからの送客読者も含む

## 収益モデル
```
challenge-tensyoku（転職）← クロスリンク
        ↓
Google検索（恋愛・婚活・マッチングアプリ関連KW）
        ↓
koi-hajime（マッチングアプリ比較）
        ↓
A8.netアフィリエイト（マッチングアプリ登録CV）
```

## サイト情報
- URL：https://koi-hajime.github.io/koi-hajime/
- GitHub：https://github.com/koi-hajime/koi-hajime
- ホスティング：GitHub Pages（公開済み）

## アフィリエイト情報
- ASP：アクセストレード（認証待ち、2026-04-05申請）
- 案件：承認待ち
- 目標単価：2,000〜5,000円/件（マッチングアプリ登録系）

## ターゲットキーワード方針
- ビッグKWは避け、属性×悩みのロングテールに集中
- 例：「フリーター 出会い マッチングアプリ」「正社員になったら 婚活」「20代 マッチングアプリ おすすめ 未経験」

## 技術仕様
- HTML / CSS のみ（バニラ）
- レスポンシブ：スマホファースト
- デザイン：challenge-tensyokuとは異なるカラー設計（暖色系・恋愛らしいトーン）
- コードには日本語コメントを入れる

## エージェント構成
- researcher.md：SEOキーワード調査・マッチングアプリ案件調査
- writer.md：マッチングアプリ比較記事の執筆
- developer.md：HTML実装・サイト更新

## 起動時の動作（重要）
このディレクトリでセッションを開始したら、必ず以下を実行すること：
1. `エージェント/leader.md` を読む
2. リーダーとして振る舞う
3. 「恋はじめのリーダーです。指示をどうぞ。」と返す
